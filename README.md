About
=======================

[LAMMJohnson](https://github.com/LAMMJohnson)'s [Fractal-Terrain-Test](https://github.com/LAMMJohnson/Fractal-Terrain-Test) ported to Javascript. Instead of using SDL, the Canvas 2D API renders the height map.
